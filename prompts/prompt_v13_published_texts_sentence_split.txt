# Dataset Instructions: Akkadian to English Translation Formatting

## Overview
These instructions describe the formatting conventions for translating Old Assyrian Akkadian texts to English, based on expert translations in the training data.

## Personal Names
- Normalize transliterated names to standard Akkadian orthography with diacritics
- Use macrons for long vowels: ā, ē, ī, ō, ū
- Use special characters: š (shin), ṣ (emphatic s), ṭ (emphatic t)
- Examples:
  - a-šùr-ma-lik → Aššur-mālik
  - im-dí-lim → Imdī-ilim
  - e-na-bé-lúm → Enna-Bēlum
  - ša-lim-a-šùr → Šalim-Aššur
  - PUZUR₄-a-šur → Puzur-Aššur

## Determinatives
Determinatives are semantic classifiers that indicate the category of the following word. They should be removed in translation:
- {d} or (d) = divine name determinative → remove, normalize the divine name
  - {d}UTU → Šamaš (sun god)
  - {d}IŠKUR → Adad (storm god)
  - {d}EN.LÍL → Illil/Enlil
- {ki} = place name determinative → remove, use place name
  - a-lim{ki} → "the City" (referring to Aššur)
  - kà-ni-iš{ki} → Kaneš
- {m} = male personal name (usually omitted in Old Assyrian)
- {f} = female personal name

## Logograms (Sumerograms)
Words written in capitals are Sumerian logograms read with Akkadian values:
- KIŠIB → "Seal of" (at start of witness/seal lists)
- DUMU → "son of" (filiation marker)
- DUMU.MUNUS → "daughter of"
- KÙ.BABBAR → "silver"
- KÙ.GI → "gold"
- KÙ.AN → "meteoric iron" (translate to English, NOT "amūtum")
- AN.NA → "tin"
- URUDU → "copper"
- TÚG → "textile(s)"
- TÚG.ḪI.A → "textiles" (plural)
- ITU / ITU.KAM → "month"
- IGI → "Witness:" or "Witnessed by" (in witness lists)
- É → "house" or "household"
- É.GAL → "palace"
- DAM.GÀR → "merchant"
- ANŠE → "donkey"
- GÚ → "talent" (unit of weight, ~30kg)

## Numbers and Units
**CRITICAL: Translate unit terms, but preserve original values.**

- **No Conversions:** Never sum up values or convert units.
  - Source: `1 ma-na 5 GÍN`
  - Correct: "1 mina 5 shekels"
  - Incorrect: "65 shekels" (Do not sum)
  - Incorrect: "1 ma-na 5 GÍN" (Do not leave untranslated)

- **Decimals:** Keep decimal notation exactly as written.
  - `0.33333 ma-na` → "0.33333 mina" (NOT "1/3 mina")

- **Unit Definitions:**
  - `ma-na` → "mina(s)"
  - `GÍN` → "shekel(s)"
  - `GÚ` → "talent(s)"


## Time Expressions
- ḫa-muš-tim / ḫa-am-ša-tim → "week(s)" (5-day Assyrian week)
- ITU.KAM → "month"
- li-mu-um → "eponymy/eponymate of" (year named after official)
- u₄-mu → "day"

## Gap and Damage Markers
**CRITICAL: Preserve gap markers in translations to maintain alignment.**
- `<gap>` in transliteration → `<gap>` in translation (single missing sign)
- `<big_gap>` in transliteration → `<big_gap>` in translation (larger lacuna)
- Gap markers MUST be parallelized between transliteration and translation
- Example: "a-na <gap> qí-bi-ma" → "To <gap> speak:"

## Standard Formulas

### Letter Opening Formula
**Prefer "From [Sender] to [Recipient]:" format for both patterns:**

1. "um-ma [Sender]-ma a-na [Recipient] qí-bi-ma"
   → "From [Sender] to [Recipient]:"

2. "a-na [Recipient] qí-bi-ma um-ma [Sender]-ma"
   → "From [Sender] to [Recipient]:"

Note: "Thus" is acceptable only when introducing direct speech within the body of a letter, not for the opening formula.

### Seal Formula
"KIŠIB [Name] DUMU [Father's name]"
→ "Seal of [Name] son of [Father's name]"

### Debt/Loan Formula
"X [amount] [commodity] i-ṣé-er [Debtor] [Creditor] i-šu"
→ "[Debtor] owes [amount] [commodity] to [Creditor]"

### Witness Formula
"IGI [Name]" → "Witness: [Name]" or "Witnessed by [Name]"

### Payment Terms
"a-na X ḫa-am-ša-tim i-ša-qal"
→ "he will pay in X weeks"

"šu-ma lá iš-qúl ... ṣí-ib-tám ú-ṣa-áb"
→ "If he does not pay (on time), he will add interest..."

## Quotation and Speech
- um-ma [Speaker]-ma → "[Speaker] said:" (in body text, not letter openings)
- Direct speech can use quotation marks or be introduced with colon
- Nested quotes: use single quotes inside double quotes

## Punctuation Guidelines
- Use colons after speaker attribution: "Šalim-Aššur said:"
- Use periods for sentence endings
- Use commas for lists and clause separation
- Use semicolons for closely related independent clauses
- Parenthetical notes: use parentheses for clarifications, e.g., "(on time)"

## Word Order
- Akkadian is verb-final (SOV), but translate to natural English word order
- Preserve emphasis when marked in original

## Quality Terms
Use standard quality descriptors:
- "good quality" (NOT "fine quality")
- "good silver" (NOT "fine silver")
- "ordinary quality" for ša qātim

## Technical Terms
Translate Akkadian terms to English like experts do:
- KÙ.AN / amūtum → "meteoric iron" (NOT kept as "amūtum")
- ṭuppum → "letter" or "tablet"
- tamkārum → "merchant"
- kutānum → "kutānum-textile" (specific textile type, keep transliterated)
- pirikannum → "pirikannum-textile" (specific textile type)

## Quality Notes
- Maintain consistency in name spellings throughout a text
- Cross-reference related clauses for coherent translation
- Preserve legal/commercial precision in contracts
